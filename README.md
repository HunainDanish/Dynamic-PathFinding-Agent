
# Dynamic Pathfinding Agent

A real-time pathfinding visualizer built with Python and Pygame. It supports A* and Greedy Best-First Search with dynamic obstacle re-planning.

## Requirements
- Python 3.x
- Pygame

## Installation
pip install pygame

## How to Run

Download main.py.

Run: python main.py

Input your desired grid size and density in the popup.

## Controls

Space: Start Search

A: Toggle Algorithm

H: Toggle Heuristic

D: Toggle Dynamic Mode

R: Randomize Grid

Left Click: Place Walls manually
